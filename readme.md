# 🔐 Key Transfer Server (Go)

Простой и безопасный сервер на Go для **одноразовой передачи ключей шифрования** между пользователями.

## 🚀 Возможности

* 🔑 Отправка ключа с генерацией одноразового токена
* 📥 Получение ключа по токену
* ⏳ Временное хранение (TTL)
* ❌ Удаление ключа после первого получения
* 🔒 HTTPS (TLS)
* 🐳 Поддержка Docker

---

## 🧠 Как это работает

1. Отправитель отправляет ключ на сервер
2. Сервер возвращает одноразовый `token`
3. Получатель использует `token`, чтобы получить ключ
4. После получения ключ удаляется

---

## 📦 API

### ➤ Отправка ключа

**POST** `/send`

#### Request:

```json
{
  "key": "MY_SECRET_KEY"
}
```

#### Response:

```json
{
  "token": "random_generated_token"
}
```

---

### ➤ Получение ключа

**GET** `/get?token=...`

#### Response:

```json
{
  "key": "MY_SECRET_KEY"
}
```

---

## 🔐 Безопасность

* Ключи хранятся **только в памяти**
* Удаляются после первого чтения
* Есть TTL (по умолчанию 5 минут)
* Используется HTTPS

> ⚠️ ВАЖНО: сервер видит ключ в открытом виде.
> Для максимальной безопасности рекомендуется использовать **end-to-end шифрование**.

## 🛠️ Локальный запуск

### 1. Сгенерировать TLS сертификаты

```bash
openssl req -x509 -newkey rsa:4096 \
  -keyout key.pem \
  -out cert.pem \
  -days 365 \
  -nodes
```

---

### 2. Запустить сервер

```bash
go run main.go
```

Сервер будет доступен на:

```
https://localhost:8443
```

---

## 🐳 Docker

### Сборка образа

```bash
docker build -t key-server .
```

---

### Запуск

```bash
docker run -p 8443:8443 key-server
```

---

### 🔐 Рекомендуемый запуск (с volume)

```bash
docker run -p 8443:8443 \
  -v $(pwd)/certs:/app \
  key-server
```

---

## 🧪 Примеры использования

### Отправка ключа

```bash
curl -k -X POST https://localhost:8443/send \
  -H "Content-Type: application/json" \
  -d '{"key":"MY_SECRET_KEY"}'
```

---

### Получение ключа

```bash
curl -k "https://localhost:8443/get?token=YOUR_TOKEN"
```
